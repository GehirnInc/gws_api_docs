Gehirn DNS
==========

Gehirn DNS は、\ `ゲヒルン`_\ が提供する高機能な DNS ホスティングサービスです。
Gehirn DNS の特徴やご利用料金などについては `Gehirn DNS サービスページ <Gehirn DNS_>`_ をご覧ください。
