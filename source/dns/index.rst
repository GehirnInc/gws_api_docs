Gehirn DNS
==========

このドキュメントでは `Gehirn DNS`_ が提供する API のご利用方法を説明します。

Gehirn DNS は、\ `ゲヒルン`_\ が提供する高機能な DNS ホスティングサービスです。
Gehirn DNS の特徴やご利用料金などについては `Gehirn DNS サービスページ <Gehirn DNS_>`_ をご覧ください。

Gehirn DNS の API ドキュメントは `OpenAPI 形式 <https://api.gis.gehirn.jp/dns/v1/openapi.json>`_ でも提供しています。
